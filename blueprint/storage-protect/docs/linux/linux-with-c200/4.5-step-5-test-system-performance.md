### Step 5: Test system performance

After the server and storage are configured, run the IBM Storage Protect workload simulation tool to verify that the medium configuration meets the expected throughput.  The tool simulates backup workloads and reports ingestion rate and resource utilisation.

1. Copy the workload simulation package from the blueprint distribution to the server.
2. Edit the simulation configuration file to specify the number of client threads, total data size and file system mount points.
3. Run the simulation and monitor CPU, memory and disk I/O usage.  The ingestion rate should fall within the **medium range (10 – 30 TB per day)**.  If the throughput is lower than expected, check network and multipath settings.
4. Review the results in ["Appendix A - Performance results"](appendix-a-performance-results.md).
