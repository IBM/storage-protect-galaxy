## About this document

This information is intended to facilitate the deployment of an IBM Storage Protect server by using detailed hardware specifications to build a system and automated scripts to configure the software. To complete the tasks, you must have an understanding of IBM Storage Protect and scripting.

This document is an addendum to the previous [IBM Storage Protect Blueprint for Linux](https://www.ibm.com/support/pages/system/files/inline-files/srv_blueprint_xlinux_v51.pdf), wherein the medium-scale system uses [IBM Storage FlashSystem C200](https://www.ibm.com/products/flashsystem-C200).
