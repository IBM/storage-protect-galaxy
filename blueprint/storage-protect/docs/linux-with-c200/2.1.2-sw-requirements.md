#### 2.1.2 Software requirements

You must install the Red Hat Enterprise Linux operating system and the IBM Storage Protect server and backup-archive client.

The following versions are required:

* Red Hat Enterprise Linux Version 9.2 or later.
* IBM Storage Protect V8.1.27 or later.
* IBM Storage Protect V8.1.25 or later backup-archive client.
* A licensed version of IBM Storage Protect is required to run the Blueprint configuration script. To obtain critical fixes, install IBM Storage Protect V8.1.27 or a later level. At the time of publication, the latest level of IBM Storage Protect was V8.1.27.
* The Blueprint configuration script V5.1 or later.
